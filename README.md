# TabZen – Intelligent Tab Organizer & Focus Assistant

TabZen automatically organizes your chaotic tabs into smart groups, reduces tab overload, and helps you stay focused without losing anything.

## Features Currently Implemented

- 🪄 **Auto-Grouping Engine:** Automatically groups your tabs as you browse into categories like "Dev Work", "Shopping", "Research", and "Entertainment".
- 💾 **Session Manager:** Save your active tabs as named sessions. Restore them entirely in a new window with one click.
- 🧹 **Duplicate Cleaner:** Detects and closes duplicate tabs to save RAM and mental load.
- 🔍 **Smart Search:** Quickly filter through all open tabs right from the popup.
- 🌓 **Beautiful UI:** A fluid, modern Glassmorphism-style popup with dark and light mode.
- ⌨️ **Keyboard Shortcuts:** Use `Ctrl+Shift+Z` to instantly "Zen" the active tab.

## Installation

### For Developers (Current State)
1. Clone this repository to your local machine.
2. Open Google Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click **Load unpacked** and select the directory containing this code.
